# Digit-Guesser

GUI that uses AI to predict a drawn digit.

## How to use
git clone 
```bash
git clone git@github.com:LucasColas/Digit-Guesser.git
```
Go to the directory
```bash
cd Digit-Guesser
```
Install the requirements (with pip for exemple) 
```bash
pip install -r requirements.txt
```

and then run the code : 
```bash
python main.py
```

Draw a digit

press enter
